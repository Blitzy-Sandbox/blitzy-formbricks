<Note>
  The Attribute Class endpoints are part of the v1 Management API. For new integrations, consider using the Contact Attribute Keys endpoints (`/api/v1/management/contact-attribute-keys`) which provide equivalent functionality for managing contact attributes. The request body accepts `name`, `key`, `type` (custom or automatic), and optional `description` fields.
</Note>
