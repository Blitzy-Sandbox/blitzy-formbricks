</Warning>

## Endpoint Details

This endpoint retrieves all identified people from your Formbricks environment. The response includes pagination support for large datasets.

### Authentication

This endpoint requires a valid API key passed via the `x-api-key` header with appropriate read permissions. See [Generate API Key](/api-reference/generate-key) for setup instructions.

### Pagination

The response includes pagination metadata. Use the returned offset and limit values to paginate through large result sets.
