To test if your API key is working, you can use the following request:

## Mandatory Headers

| Name            | x-api-key                |
| --------------- | ------------------------ |
| **Type**        | string                   |
| **Description** | Your Formbricks API key. |

## Request

```bash cURL (Cloud)
GET - /api/v1/management/me

curl --location \
'https://app.formbricks.com/api/v1/management/me' \
--header \
'x-api-key: <your-api-key>'
```

```bash cURL (Self-Hosted)
GET - /api/v1/management/me

curl --location \
'https://your-formbricks-instance.com/api/v1/management/me' \
--header \
'x-api-key: <your-api-key>'
```

## Response

<Note>
  The `/me` endpoint returns information about the environment and project associated with your API key. If your key has access to multiple environments, the response reflects the first matching environment with the required permission level.
</Note>
